# Finance Dashboard UI

## Overview
A responsive finance dashboard built using React and Tailwind CSS to track transactions, visualize spending, and generate insights.

## Features
- Dashboard with balance, income, expenses
- Charts (trend + category breakdown)
- Transactions table with search, filter, sorting
- Role-based UI (Viewer/Admin)
- Insights and analytics
- Dark mode

## Tech Stack
- React (Vite)
- Tailwind CSS
- Recharts
- Context API + useReducer

## Setup
npm install
npm run dev

## Role Switching
Use the dropdown in the header to switch roles.
